# Lógica de Programación 3 – Factorial

Este proyecto es una práctica básica de lógica de programación en JavaScript.
El objetivo es calcular el factorial de un número entero ingresado por el usuario,
aplicando validaciones y estructuras de control.

## ¿Qué hace el programa?

- Solicita al usuario un número entero mediante un input.
- Valida que el valor ingresado sea numérico y sin decimales.
- Calcula el factorial del número ingresado.
- Muestra el resultado o un mensaje de error en pantalla.

## Tecnologías usadas

- HTML
- Bootstrap
- JavaScript

## Ejemplos

- Entrada: 5  
  Salida: 120  

- Entrada: 6  
  Salida: 720  

## Estructura

- index.html: Interfaz y entrada de datos.
- factorial.js: Lógica y validaciones del cálculo.
